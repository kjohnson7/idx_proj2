Interactive Elements

The controls below the chart allow me to get a better sense of my time for future planning. The controls have a mouseover state in which the cursor changes to a hand in order to show that it is an interactive element. Once the control is selected, the label turns gray to show that it is no longer selected nor a part of the visualization.

Hover States

After discovering what my week would look like if I cut out sleep, I found myself explaining what I was actually doing during each of the scheduled blocks. Therefore, I created a hover state to allow people to get more details about my time while moving the cursor over different blocks. The hover states shows people exactly which block they are hovering over by changing the color of the active block.
